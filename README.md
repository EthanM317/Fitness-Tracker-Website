# Fitness-Tracker-Website
A fitness tracker website for CSCI 3230U

## Setup
1. Install dependencies for both client and server:
   ```
   cd client && npm install --legacy-peer-deps
   cd ../server && npm install
   ```

2. Start the server:
   ```
   cd server && npm start
   ```

3. Start the client:
   ```
   cd client && npm run dev
   ```
